# coreyeudell.com

Personal portfolio and landing page. Built with Next.js, TypeScript, and Tailwind CSS.

## Getting Started

```bash
npm install
npm run dev
```

Visit [http://localhost:3000](http://localhost:3000).

## Tech Stack

- **Framework:** Next.js 15 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS 4
- **Database:** SQLite + Drizzle ORM
- **Deployment:** Vercel

## Project Structure

```
src/
├── app/            # Pages, layouts, API routes
│   ├── api/        # Backend endpoints
│   └── ...         # Page routes
├── components/     # Reusable UI components
├── lib/            # Utilities, DB client, config
└── data/           # Static content and project data
```
